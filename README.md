# 🔌 App para Análise de Superfícies Equipotenciais

Aplicativo simples para visualização de **superfícies equipotenciais** e **linhas de campo elétrico** em diferentes configurações de condutores.

## 📘 Recursos
- Visualização de equipotenciais  
- Linhas de campo elétrico  
- Configurações básicas de condutores
